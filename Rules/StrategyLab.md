# Strategy Lab

Version: v1.0.1-basic

## 新策略加入前必問

1. 它有沒有邏輯？
2. 它適合什麼市場？
3. 它適合 Louis 嗎？
4. 它是否提升決策品質？
5. 是否需要 Replay 驗證？

## 策略處理流程

新策略
-> Strategy Lab 分析
-> Replay 測試
-> Review
-> 決定是否加入 LIOS

## 範例：投信 + MACD + KD 策略

納入：
- 投信持續買超：加分項
- MACD 翻紅：趨勢確認
- KD 黃金交叉：輔助進場

不納入：
- 股價必須低於100元
- KD > 90 必賣

## 原則

任何規則不能因一兩次成功就加入，也不能因一兩次失敗就刪除。
