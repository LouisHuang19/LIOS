# Replay Checklist

Version: v1.0.6-basic

## 開始前

- [ ] 是否明確定義 Replay 日期？
- [ ] 是否只使用當時可見資訊？
- [ ] 是否有 K 線與主要指標？
- [ ] 是否有大盤背景？
- [ ] 是否記錄資料完整度？

## Decision Engine

- [ ] 是否記錄 Decision Score？
- [ ] 是否記錄燈號？
- [ ] 是否做反方檢查？
- [ ] 是否檢查 Kill Switch？
- [ ] 是否說明進場 / 不進場原因？

## 進場後

- [ ] 是否逐日判斷？
- [ ] 是否記錄出場訊號？
- [ ] 是否避免偷看未來？
- [ ] 是否以 LIOS 出場，不是用最高價出場？

## 結束後

- [ ] 是否計算實際報酬？
- [ ] 是否分類 Perfect / Good / Neutral / Fail？
- [ ] 是否記錄 Lessons Learned？
- [ ] 是否判斷是否納入 Replay30 統計？
