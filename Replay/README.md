# Replay System

Version: v1.0.6-basic
Compatible: LIOS v1.0.5-basic+
Status: Active

## Purpose

Replay System 是 LIOS 的回測室，用來驗證：

> 如果當時只使用當時能看到的資料，LIOS 會不會做出正確的進出場？

## Core Rule

Replay 不看最高價。

Replay 只看：

1. LIOS 何時進場
2. LIOS 何時出場
3. 實際報酬率
4. 是否符合紀律
5. 是否需要修正規則
