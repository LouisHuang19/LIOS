# Replay Examples

Version: v1.0.6-basic

## Example: 南亞科

錯誤寫法：

> 後來漲到 505，所以 6/15 買是成功。

正確寫法：

1. 6/15 當天 LIOS 是否給進場訊號？
2. 若進場，進場價是多少？
3. 之後哪一天出現 LIOS 出場訊號？
4. 出場價是多少？
5. 實際報酬率是多少？

## Example: 華旭先進

重點不是最低點在哪裡。

重點是：

- LIOS 是否避免在弱勢中攤平？
- 是否等待 RS / OBV / MACD 改善？
- 若仍進場，結果如何？

## Example: 台積電

核心股不應因 KD 過熱單獨賣出。

Replay 要檢查：

- 基本面是否變化？
- RS 是否仍強？
- OBV 是否持續？
- 是否只是短線波動？
