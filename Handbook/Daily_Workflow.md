# Handbook — Daily Workflow

Version: v1.0.8-basic

## 平日 5 分鐘流程

1. 啟動 LIOS
2. 查看風險室
3. 查看投資室
4. 若沒有交易需求，結束
5. 若有交易需求，跑 Decision Engine

## 盤中原則

盤中只觀察，不急著修改規則。

## 收盤後流程

收盤後適合：

- 更新 Portfolio
- 看戰情室
- 記錄 Journal
- 決定是否加入 Replay 候選

## 每週流程

1. 檢查弱勢持股
2. 檢查現金比例
3. 檢查 Watchlist
4. 選一個 Replay 案例

## 每月流程

1. Review Lab
2. Replay 統計
3. DNA Evolution
4. 決定是否需要 patch
