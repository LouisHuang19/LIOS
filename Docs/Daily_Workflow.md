# Daily Workflow

Version: v1.0.2-basic

## 目的

固定每天使用 LIOS 的順序，避免每次重新想流程。

## 每日最小流程

1. 啟動 LIOS
2. 查看風險室
3. 查看投資室
4. 若沒有交易需求，結束
5. 若有交易需求，跑 Decision Engine
6. 若有新策略，送 Strategy Lab
7. 若有歷史案例，送 Replay Lab

## 開盤前

可看：
- 投資室
- 風險室
- 行事曆
- 昨日警報

原則：
- 不因盤前情緒追單
- 若風險室黃燈以上，提高買進標準

## 收盤後

可看：
- 戰情室
- 持股診斷
- Journal
- Replay 候選案例

收盤後適合做決策，盤中適合觀察。

## 每週流程

1. 更新持股健康度
2. 檢查風險室燈號
3. 檢查弱勢股是否需要處理
4. 檢查是否有 Replay 案例可加入
5. 檢查 Strategy Lab 是否有新策略待驗證

## 每月流程

1. Review Lab
2. Performance Center
3. Replay 統計
4. DNA Evolution
5. ChangeLog 檢查
6. 決定是否需要版本更新

## 禁止事項

- 不因單日大漲直接新增規則
- 不因單日大跌刪除規則
- 不因成本價而攤平弱勢股
- 不在風險室橘燈或紅燈時追高
