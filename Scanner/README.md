# Scanner System

Version: v1.0.7-basic
Compatible: LIOS v1.0.6-basic+
Status: Active

## Purpose

Scanner System 是 LIOS 的選股室，用來篩選符合 Louis 投資系統的標的。

它不是每天找明牌，也不是找漲停股。

它回答：

> 哪些股票值得進一步研究？哪些股票可以替代弱勢持股？哪些股票即將從觀察轉為可布局？
