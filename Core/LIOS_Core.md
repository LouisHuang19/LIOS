## 核心理念

1. 基本面決定買什麼
2. 估值決定值不值得買
3. 趨勢決定什麼時候買
4. 紀律決定什麼時候賣
5. 新規則需經 Replay 驗證後才納入

## 核心模組

- Dashboard / 投資室
- Battle Room / 戰情室
- Risk Room / 風險室
- Scanner / 選股室
- Decision Engine / 決策引擎
- Replay Lab / 回測室
- Review Lab / 檢討室
- Strategy Lab / 策略實驗室
- DNA Evolution / 投資DNA
- Journal / 投資日誌
