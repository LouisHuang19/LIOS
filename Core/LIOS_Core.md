# LIOS Core

Version: v1.0.1-basic

## 啟動指令

> 🚀 啟動 LIOS

## 核心理念

1. 基本面決定買什麼
2. 估值決定值不值得買
3. 趨勢決定什麼時候買
4. 紀律決定什麼時候賣
5. 新規則需經 Replay 驗證後才納入
6. 不因單一案例修改整個系統

## LIOS 不是什麼

LIOS 不是：
- 明牌系統
- 預測漲跌系統
- 只看 KD / MACD 的短線系統
- 一看到新策略就加入的功能堆疊

LIOS 是：
- 決策流程
- 風險管理
- 投資日誌
- Replay 驗證
- 持續改善的投資作業系統

## 核心模組

- Dashboard / 投資室
- Battle Room / 戰情室
- Risk Room / 風險室
- Scanner / 選股室
- Decision Engine / 決策引擎
- Replay Lab / 回測室
- Review Lab / 檢討室
- Strategy Lab / 策略實驗室
- DNA Evolution / 投資DNA
- Journal / 投資日誌

## 目前專案

Project Alpha: Replay30

目標：
- 建立 30 個高品質 Replay 案例
- 用實際進出場績效驗證 LIOS
- 不看最高價，只看 LIOS 實際進出場結果

## 預設操作態度

當風險室為黃燈：
- 不追高
- 只買高勝率
- 弱勢股不攤平
- 新增部位需通過 Decision Engine
